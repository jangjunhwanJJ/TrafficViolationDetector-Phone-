# TrafficViolationDetector (Android)

블랙박스 영상에서 교통법규 위반을 실시간으로 감지하는 Android 앱

---

## 기능

- 카메라 실시간 영상 분석 (CameraX)
- 3가지 위반 유형 감지: **신호위반 / 중앙선침범 / 진로변경위반**
- 위반 감지 시 전후 8초 영상 자동 저장 (갤러리)

---

## 모델 구성

| 모델 | 형식 | 역할 |
|------|------|------|
| LRCN | TFLite fp16 | 25프레임 시퀀스 → 위반 유형 분류 |
| YOLOv5 x3 | ONNX | 위반 유형별 객체 탐지 (확인용) |
| YOLO26n-seg | ONNX | 차선 탐지 (정지선/중앙선/백색차선) — 통합 예정 |

> **모델 파일은 용량 문제로 레포에 포함되지 않습니다.**
> 아래 링크에서 다운로드 후 `app/src/main/assets/` 에 넣어주세요. (링크 추후 추가)

---

## 감지 파이프라인

```
카메라 프레임
    │
    ├─► VideoRecorder   원형버퍼(5초) 유지, 위반 시 앞5초+뒤3초 저장
    │
    └─► ViolationDetector
            ├─ LRCN          25프레임 시퀀스 → 위반 유형 분류
            ├─ YOLOv5        위반 유형별 객체 탐지로 LRCN 결과 확인
            └─ 보정 필터      정지선/신호등/HSV/모션 기반 오탐 방지
```

### 문제점
- [ ] **오탐지 많음** — LRCN이 정상 주행 클래스 없이 3개 위반 유형만 학습됨. 어떤 영상이든 3개 중 하나로 분류함
- [ ] **Detectron2 차선 탐지 미통합** — 기존 PC용 Detectron2 모델은 Android 배포 불가. YOLO26n-seg로 교체 학습 중
- [ ] **카메라 프리뷰 시각화 없음** — 탐지 결과를 화면에 오버레이하는 기능 미구현
